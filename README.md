# Site Psicólogo - Luiz Felipe Braziliano

Este projeto é o site profissional do psicólogo Luiz Felipe Braziliano, desenvolvido com React, TypeScript e Tailwind CSS.

## Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Rode o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

3. Acesse `http://localhost:5173` (ou a porta indicada no terminal).

## Tecnologias

- **React**
- **TypeScript**
- **Tailwind CSS**
- **Framer Motion**
- **Lucide React**
